eggdropgoez

auto install IRC BOT eggdrop with goez just for fun :D

Install eggdrop goez

Ambil script nya dengan cara: 
wget eggo.ga/eggoez.sh
Lalu ganti izin (permission) 
chmod +x eggoez.sh
Jalankan Installasi dengan cara 
./eggoez.sh
Dan silahkan menjawab pertanyaan nya ;)

Catatan!

Sumber eggdrop diatas adalah patch. Sebagian module telah dihapus dan di sesuaikan. Jika anda tidak ingin versi patch, silahkan coba wget versi latest dan orginalnya yang mendukung SSL & IPv6 dengan perintah di bawah ini.

Hanya mengganti sumber

wget eggo.ga/meggoez.sh && chmod +x meggoez.sh && ./meggoez.sh

Untuk eggdrop IPv6

wget eggo.ga/eggoez6.sh && chmod +x eggoez6.sh && ./eggoez6.sh
Dan cobalah untuk menjawab pertanyaan nya ;)
Pastikan tool untuk compile, c++, make dll untuk eggdrop bot sudah terpasang pada server anda, untuk lebih jelasnya baca tutorial memulai eggdrop.
