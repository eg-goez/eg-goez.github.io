Pertanyaan yang akan ditemui

Setelah anda menjalankan file sh anda akan menemui pertanyaan yang dibutuhkan untuk menjalankan eggdrop

Yakin untuk memulai?

Silahkan jawab ya/y jika anda ingin memulai atau jawab tidak/t jika anda ingin membatalkan.

Jika anda menjawab ya/y anda akan menemui pertanyaan yang harus anda jawab karena ini dibutuhkan untuk pembuatan konfigurasi BOT anda.

1) Nick Owner Bot

Nick owner bot adalah nick anda, nick yang anda masukkan akan menjadi pemilik (owner) BOT tersebut.

2) Server Tujuan

Masukkan host/ip server yang akan di tuju BOT (ini juga berarti server akan di gunakan BOT). Untuk Standar BOT ini menggunakan Port 6667 Namun jika Server port anda berbeda misalkan SSL atau tambahan Password Server gunakan tanda : sebagai pemisah. Misal irc.eggoez.com:+9999:goez

3) Listen Telnet

Isikan port Telnet yang ingin anda pakai, ini berfungsi untuk men DCC bot, jika bisa masukkan antara 1024-65535

4) IP Shell / VPS

Gunakan ip vps anda agar bot lekas terhubung, default untuk hostname adalah localhost dan ip 127.0.0.1

5) Nick Bot yang anda inginkan

Nick bot yang ingin anda gunakan, jangan gunakan symbol atau karakter unik yang tidak di dukung di jaringan anda.

6) Nick Alternatif Bot yang anda inginkan

Nick ke dua atau alternatif nick bot jika real nick bot sudah ada / digunakan user lain.

7) Username bot?

Username bot yang ingin anda gunakan, jangan gunakan karakter yang tidak di dukung.

8) Base Channel Bot

Ini adalah Channel BOT, ketika bot berhasil dijalankan maka bot akan menuju channel ini, dan ini adalah base (terkunci) tidak bisa part/left.

9) IPV6/Vhost

IPV6/Vhost yang digunakan oleh BOT, ini jika anda mempunyai konfigurasi IPv6 sendiri pada server anda, namun jika anda tidak tahu, silahkan Enter (kosongi saja).
